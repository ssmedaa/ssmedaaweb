P R O J E C T S

These are some university EECS course projects that Samra has done, but also some indepent projects.



Minesweeper Game

Python program game that allows the player to pick a coordinate point on the game board displayed.
The program game board creates random amount of mines in different locations and will check if the players coordinate point.
The game will continue if player is safe from mine, but the program will end once a spot containing a mine is choosen.

Student studentTranscript

C program to allow the student (user) to: sort/clear database, load/ write/ empty disk, view courses, remove/swap record, and
display database.The program uses pointers to store courses and to allow for changes to occur.

Passport

Unix shell program to allow users to input their passport informations. This program will check if the inputted information follows
the guidelines and then stores. Otherwise, the program will ask for passport information again.
